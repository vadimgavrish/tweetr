# Grasshopper Project

Grasshopper is a simple, single-page Twitter clone.

## Dependencies

- Node.JS
- body-parser
- chance
- express
- moment
- md5
- mongo
- mongodb

## Getting Started

- Install all dependencies (using the `npm install` command)
- Run the app from your teminal within the main folder using the `run npm local` command
